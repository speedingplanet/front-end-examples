Different kinds of anchors

Anchors to different pages

  • Relative link (no starting /):
    <a href="line-breaks.html">Line Breaks</a>
    Line Breaks
  • Relative link (with path info):
    <a href="nested/path.html">Path</a>
    Path
  • Relative link (up and out of this directory):
    <a href="../css/index.html">CSS</a>
    CSS
  • Absolute link (starts with a /):
    <a href="/html/nested/path.html">Absolute reference</a>
    Absolute reference
  • External link (starts with a protocol, like http:// or https://):
    <a href="https://intuit.com">Intuit</a>
    Intuit
